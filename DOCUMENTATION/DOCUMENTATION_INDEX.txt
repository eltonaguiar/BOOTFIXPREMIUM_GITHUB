SECTION 4: EXECUTABLE COMPONENTS
───────────────────────────────────────────────────────────────────────────────────

PRODUCTION APPLICATION:

MiracleBoot.ps1 (Main Application) ✓ PRODUCTION READY
  → Primary entry point
  → Contains main recovery logic
  → Handles environment detection
  → Routes to GUI or TUI based on OS

RunMiracleBoot.cmd (Batch Launcher) ✓ PRODUCTION READY
  → Windows batch file launcher
  → Handles PowerShell execution
  → User-friendly for non-technical users
  → Error handling for missing PowerShell

SUPPORT MODULES (All ✓ PRODUCTION READY):

WinRepairCore.ps1
  → Core repair functions
  → BCD operations
  → Boot sector management
  → Repair logic foundation

WinRepairTUI.ps1
  → Text-based user interface
  → Menu system for WinPE/WinRE
  → Command-line friendly
  → Works in minimal recovery environments

WinRepairGUI.ps1
  → Modern Windows Presentation Foundation (WPF) interface
  → Full Windows only
  → Tab-based navigation
  → Advanced diagnostics display

Harvest-DriverPackage.ps1
  → Driver extraction from running system
  → Creates portable driver packages
  → Organizes by hardware type
  → Generates inventory CSV

NetworkDiagnostics.ps1
  → Network connectivity testing
  → Adapter status detection
  → Driver discovery
  → Troubleshooting tools

Diskpart-Interactive.ps1
  → User-friendly disk management
  → Safe wrapper around diskpart.exe
  → Auto-detection of Windows volume
  → Educational prompts

Generate-BootRecoveryGuide.ps1
  → Creates SAVE_ME.txt FAQ
  → 3000+ word troubleshooting guide
  → Step-by-step recovery instructions
  → Error code reference

KeyboardSymbols.ps1
  → Special character input helper
  → Alt code reference
  → Symbol database
  → Useful for command-line input

TestRecommendedTools.ps1
  → Validates recommended tools feature
  → Tests GUI/TUI integration
  → Verifies XAML syntax
  → Checks feature completeness
