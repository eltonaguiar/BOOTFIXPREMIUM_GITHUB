  4. Output automatically logged

  OPTION 2: RIGHT-CLICK METHOD
  ────────────────────────────
  1. Right-click MiracleBoot.ps1
  2. Select: "Run with PowerShell as Administrator"
  3. Accept UAC prompt
  4. MiracleBoot launches

  OPTION 3: POWERSHELL METHOD
  ──────────────────────────
  1. Press Win+X, select "PowerShell (Admin)"
  2. cd to MiracleBoot folder
  3. Run: powershell -NoProfile -ExecutionPolicy Bypass -File "MiracleBoot.ps1"
